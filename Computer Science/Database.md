
> [!def] datatype
> A data type classifies how the data is stored, displayed and the operations that can be performed on the stored value.

A **database** is an organized collection of data that is stored and accessed electronically. It allows for efficient storage, retrieval, and management of information.

A **field** is a single piece of data or attribute in a record. It represents one category of information, such as a name, date of birth, or ID number.

A **record** is a complete set of fields related to one entity or item in the database. For example, one person's information in a student database is a record.

A **primary** **key** is a unique identifier for each record in a database table. It ensures that no two records have the same value in the primary key field.
