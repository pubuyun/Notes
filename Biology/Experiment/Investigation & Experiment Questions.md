> to find out the effect of duration of jogging on heart rate

*Independent Variable*: duration of jogging, (1min, 2min, 3min, 4min, 5min)
*Dependent Variable*: change in heart rate (bpm), measured by smartwatch
*Controlled Variable*: same temperature, same altitude, same weight, same speed of jogging...

*Methods*: 
1. use a smartwatch to measure the initial heartrate
2. jog for 1 minute and record the heartrate
3. repeat step 2 for 5 minutes
4. after resting (recover to initial heartrate), repeat step 1-3 for at least 2 times
5. record the data in the table

*Safety Precaution*: wear clothes suitable for sport, and the participants are healthy

### Errors
> Errors in procedure that can cause problem

- ***Is there color change/color observation involved?***
    - **Yes**
        - **Error:** subjective color observation
        - **Improvement:** use a **colorimeter**/white tile/color standard
            
- ***Is there a water bath involved in the procedure?***
    - **Yes**
        - **Error:** temperature is not controlled
        - **Improvement:** use a **thermostatically controlled water bath** instead
            
- ***Did you use vague measurement to add materials?***  
    (e.g., fill test tube half-full/3cm high/add 5 drops...)
    - **Yes**
        - **Error:** inaccurate measurement of volume using xxx
        - **Improvement:** use a **syringe/burette** instead
            
- ***Did you count/add drops using a drop bottle?***
    - **Yes**
        - **Error:** drop size different each time
        - **Improvement:** use a **pipette/syringe** instead
            
- ***Did the procedure tell you to do repeats?***
    - **No**
        - **Error:** no repeats
        - **Improvement:** repeat two or more trials and **take** **average**
- ***Did the procedure ask for mixing/stirring when you mix liquids?***
    - **No**
        - **Error:** xx and xx not mixed
        - **Improvement:** **stir** the mixture
- ***If tests were done simultaneously or timing was delayed:***
    - **Error:** unable to start **timing** immediately/tests not **synchronized**
    - **Improvement:**
        - Do each group **separately**
        - Have **two students** time each tube
        - Use **staggered starts**

#### Reliability
> repeatability, likelihood of getting the same results if you do it again
- To increase: **Mean**
#### Precision
> little deviance from mean result
- To increase: use more precise apparatus with smaller scale divisions
#### Accuracy
> how close the measured value is to the true value
- To increase: **reduce errors**
