## 一、医学英语构词系统
**医学词 = 前缀 + 核心词根 + 次级词根 + 后缀**

**示例：**
- **endometriosis** = endo（内）+ metr（子宫）+ osis（病变）
- **hyperechogenicity** = hyper（过多）+ echo（回声）+ gen（产生）+ ic（的）+ ity（状态）

## 📌 记忆与应用指南
1. **先拆词**：遇到生词，先按“前缀+词根+后缀”拆分。
2. **核心词根定系统**：如 metr/hyster → 子宫系统；salping → 输卵管。
3. **后缀定性质**：-itis → 炎症；-oma → 肿瘤；-rrhea → 流出。
4. **前缀定程度/方向**：hyper- → 过多；retro- → 向后。
5. **同义词根对照**：拉丁 vs 希腊（如 uter vs hyster）。
